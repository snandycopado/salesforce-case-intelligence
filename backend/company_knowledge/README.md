# Company Standard Knowledge Articles

Place your company's standard knowledge articles here.

## File Naming Convention

Name files by Case Type for automatic matching:
- `Account_Issue.md` — matched when Case Type = "Account Issue"
- `Billing.md` — matched when Case Type = "Billing"
- `Technical_Support.md` — matched when Case Type = "Technical Support"
- `General.md` — fallback, always included

Files can be `.md`, `.txt`, or `.docx` format.

## Content Guidelines

Each file should contain your company's standard procedures for that case type:
- Standard resolution workflows
- Email templates (To addresses, subject lines, body)
- Escalation contacts and procedures
- SLA requirements
- Team-specific instructions (e.g., how to contact Okta team)
